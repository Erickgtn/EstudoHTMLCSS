########### Funções basicas com arrays ##############


Funcao push ==> Adiciona um novo elemento no final do array

ex.: let arrays = ["item 1","item 2", "item 3"]

array.push("item 4")

["item 1","item 2","item 3","item 4"]

Funcao pop ==> Remove o ultimo elemento do array

ex.: let arrays = ["item 1","item 2","item 3"]

array.pop()

["item 1","item 2"]

Funcao shift ==> Remove o primeiro elemento do array

ex.: let arrays = ["item 1","item 2","item 3"]

array.shift()

["item 2","item 3"]

Funcao unshift ==> Adiciona o elemento no início do array

ex.: let arrays = ["item 1","item 2","item 3"]

array.unshift("item 0")

["item 0","item 1","item 2","item 3"]

Função que retorna o tamanho do array ( quantos elementos tem no array)

console.log(array.length)

Função que retorna o índice do elemento dentro do array

let position = array.indexOf("item 1")

console.log(position)

Obs.:Caso o elemento não esteja no array, ele retornará -1 como índice

########### Manipulando Arrays com Splice e Slice ##############

Função Splice

Sintaxe: array.splice(index[,deleteCount[,elemento1[, ...[,elementoN]]])
*Espera um primeiro parâmentro que é um índice do Array
*O segundo é aquantidade de elementos que queremos romver a partir deste índice
*Os próximos são os elementos que queremos adicionar no lugar
*Apenas o primeiro parâmetro é "Obrigatório"

Ex.
let nomes = ["Maria","Joao","Carlos","Beto"]
let removeNomes = nomes.splice(1,2,"Roberta","Joana","Lívia")
console.log(nomes)
Terá como retorno:["Maria","Roberta","Joana","Lívia","Beto"]

Função Slice

Fatiar ou dividir
Sintaxe: array.slice([begin[,end]])
*Possui um primeiro parâmetro que é o índice inical
*O segundo parâmetro é o índice final
*Retorna todos os elementos entre o lemento de índice "begin" e o anterior ao elemento "end"

Ex.
let nomes = ["Maria","Joao","Carlos","Beto"]
let extrairNome = nome.slice(1,3)
console.log(nomes)
Terá como retorno:["Roberta","Joana"]

let nomes = ["Maria","Joao","Carlos","Beto"] let upCaseNomes = nomes.map(function(teste){ let upcase = teste.toUpperCase() return upcase }) console.log("Nomes depois: "+upCaseNomes)

########### Funcao Filter ##############

Interações em Array Função Filter

let nomes = ["Maria","Joao","Carlos","Beto","Max"] let qntCaracter = nomes.filter(nome => nome.length <=3) console.log(qntCaracter)

Interações em Array Função Find

let nomes = ["Maria","Joao","Carlos","Beto","Max"] let qntCaracter = nomes.find(nome => nome.length <=3) console.log(qntCaracter)
